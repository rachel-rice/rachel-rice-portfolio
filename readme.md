# Rachel Rice's Portfolio

Rachel Rice's Portfolio is a personal website showcasing recent projects, professional information, and services offered. The site is designed to highlight work in a clean, approachable format and demonstrate design and frontend skills.

**Link to project:** https://rachelrice.netlify.app/

![Screenshot of the application](images/screenshot-landing.png)

## How It's Made:

**Tech used:** HTML, CSS, SCSS, JavaScript

The site is structured with HTML for content, SCSS and CSS for styling, and JavaScript for interactive functionality. It was built as a customizable template, which was adapted to highlight personal branding, project galleries, and informational sections about professional offerings.

The portfolio layout prioritizes clarity, readability, and responsive design, ensuring a smooth experience across desktop and mobile devices.

## Customizations

- Adapted template styling to match personal branding and project presentation  
- Added interactive elements such as navigation highlighting and project previews  
- Organized project sections for clarity and ease of use  

## Future Improvements

Future enhancements could include additional interactivity, animations, accessibility improvements, and more dynamic project filtering to better showcase work.

## Lessons Learned:

This project reinforced the value of iterative development and keeping a portfolio up-to-date. Building from a template allowed focus on content and branding while learning to adapt and customize an existing structure.
